Blood Pressure Analysis Project

Project Overview

This project investigates how physiological and lifestyle factors relate to blood pressure. The dataset is simulated for practice purposes and includes variables such as age, weight, body surface area, pulse, stress, and duration of elevated blood pressure. All analysis was performed using R Studio.

Key Takeaways:
  • Exploration and visualization help detect patterns in the data.
  • Simulated datasets are useful for testing regression models.

Exploratory Analysis

Initial analysis included summary statistics, scatterplots, and correlations to examine relationships between variables.

Key Takeaways:
  • Weight and body surface area show strong relationships with blood pressure.
  • No obvious outliers or data issues detected.

Regression Modeling

Simple and multiple linear regression models were fitted to explore how predictors affect blood pressure. Weight was identified as the strongest single predictor, and a multiple regression model including age, weight, and body surface area was selected. 


# Example: simple regression in R
model <- lm(BP ~ Weight, data = data)
summary(model)
plot(BP ~ Weight, data = data)
abline(model, col="blue")

Selected Model: Predicted BP = -13.67 + 0.70·Age + 0.91·Weight + 4.63·BSA

Key Takeaways:
  • Multiple regression improves predictive accuracy by combining key variables.
  • Weight, age, and body surface area are the most influential predictors.

Model Validation

  • Residuals indicate approximately constant variance.
  • Normality checks suggest residuals are roughly normally distributed.
  • No extreme leverage points or influential outliers detected.
Key Takeaways:
  • The selected model meets standard assumptions for linear regression.
  • Predictions are expected to be reliable within the data range.

Prediction

Using the selected model, the predicted blood pressure for a sample individual is 104.76 mmHg, with a 95% prediction interval of 103.74–105.79 mmHg.

Key Takeaways:
  • Model predictions align closely with observed data.
  • Illustrates practical application of regression modeling.
