Military Coups in Sub-Saharan Africa

This project examines how political and demographic factors are related to the number of successful military coups in sub-Saharan Africa. Variables considered include years of military oligarchy, number of political parties, population size, number of elections, regime duration, and country size. All analyses were conducted using R Studio.

Key Points:
  • Exploring the data visually and statistically helps detect relationships between variables.
  • Single-variable models show that the duration of oligarchy has the strongest influence on coups.

Initial exploration included scatterplots and boxplots to understand how each variable relates to the number of coups, as well as correlation checks to see if predictors are dependent.

Poisson regression models were used to account for count data. Single-variable models suggested that oligarchy years had the lowest AIC, indicating best fit. A full model including all predictors was also tested, followed by stepwise reduction using AIC, which resulted in a final model with oligarchy years and number of elections. 


  # Example: Poisson regression in R
  full_model <- glm(miltcoup ~ oligarchy + parties + popn + size + numelec + numregim, 
                    family = poisson, data = africa)
  reduced_model <- step(full_model, direction="both")
  summary(reduced_model)

The final model indicates that longer periods of oligarchy and a higher number of elections increase the expected number of successful coups.

Key Points:
  • The reduced model fits the data well, according to chi-squared tests.
  • No significant over- or under-dispersion was detected.
  • Each additional year of oligarchy or election is associated with an exponential increase in expected coups.
