Blood Pressure Analysis Project

Overview

This project explores how physiological and lifestyle factors relate to blood pressure. Simulated data includes variables such as age, weight, body surface area, pulse, and stress levels. The analysis was performed using R Studio.

Exploratory Analysis

Initial exploration included summary statistics, scatterplots, and correlations to detect patterns.

Modeling

Simple and multiple regression models were used to understand how different predictors affect blood pressure. Weight was the strongest single predictor, and a multiple regression model including age, weight, and body surface area was selected. 


# Example: fitting a simple regression in R
model <- lm(BP ~ Weight, data = data)
summary(model)
plot(BP ~ Weight, data = data)
abline(model, col="blue")

Selected Model: Predicted BP = -13.67 + 0.70·Age + 0.91·Weight + 4.63·BSA

Model Validation

  • Residuals show approximately constant variance.
  • Residuals are roughly normally distributed.
  • No extreme leverage or influential points detected.

Prediction

Using the selected model, the predicted blood pressure for a sample individual is 104.76 mmHg, with a 95% prediction interval of 103.74–105.79 mmHg.
