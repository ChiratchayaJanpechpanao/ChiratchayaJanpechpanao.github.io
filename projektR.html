Blood Pressure Analysis Project

Overview

This project explores relationships between physiological and lifestyle factors and blood pressure using R Studio. The dataset is simulated and includes variables such as age, weight, body surface area, pulse, stress, and duration of elevated blood pressure.

Main goals:

  • Identify key factors affecting blood pressure.
  • Build simple and multiple regression models.
  • Validate model assumptions and check reliability.
  • Make predictions for new observations.

Exploratory Analysis

Initial exploration included summary statistics, scatterplots, and correlations to observe trends and detect potential relationships.

Modeling

Simple linear regressions were fitted for each predictor. Weight emerged as the strongest single predictor of blood pressure. A multiple regression model including age, weight, and body surface area was selected based on statistical criteria.

Selected Model: Predicted BP = -13.67 + 0.70·Age + 0.91·Weight + 4.63·BSA

Model Validation

  • Residuals indicated constant variance.
  • Normality checks suggested residuals are approximately normal.
  • No extreme leverage points or influential outliers were detected.

Prediction

Using the selected model, the predicted blood pressure for a sample individual is 104.76 mmHg. The 95% prediction interval is 103.74–105.79 mmHg, indicating a precise model fit.
