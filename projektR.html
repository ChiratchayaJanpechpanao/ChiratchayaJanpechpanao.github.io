Blood Pressure Analysis Project

Project Overview

This project explores the relationships between several personal and lifestyle factors and blood pressure levels using R Studio. The dataset contains simulated data on age, weight, body surface area, duration of high blood pressure, pulse, and stress levels.

The main goals were to:

  • Investigate which factors may influence blood pressure.
  • Compare different regression models to identify key predictors.
  • Check model assumptions and assess reliability.
  • Use the model to make predictions for new observations.

Exploratory Data Analysis (EDA)

Preliminary exploration included summary statistics, density plots, scatterplots, and pairwise correlations to detect patterns or potential correlations. 


# Load and inspect data
data <- read.table('blood.txt', header = TRUE)
str(data)

# Basic EDA
plot(density(data$BP), main="Blood Pressure Density")
pairs(data[, -1], main="Scatterplots of Variables")
cor(data[, -1])

Regression Modeling

Simple linear regression models were fitted for each predictor. Among them, weight appeared to be the most influential variable. Multiple regression models were then fitted to explore combined effects. 


# Example: simple regression with weight
model_weight <- lm(BP ~ Weight, data = data)
summary(model_weight)
plot(BP ~ Weight, data = data)
abline(model_weight, col="blue")

Model Selection

Bayesian Information Criterion (BIC) was used to select a reduced model with three predictors: age, weight, and body surface area. This model balances complexity and explanatory power.

Predicted BP = -13.67 + 0.70·Age + 0.91·Weight + 4.63·BSA

Model Validation

  • Residuals indicated approximately constant variance.
  • Normality checks showed residuals were roughly normally distributed.
  • Leverage and influence diagnostics suggested no extreme outliers.
  • Shapiro-Wilk and Durbin-Watson tests confirmed model assumptions were acceptable.

Prediction Example

Using the selected model, the predicted blood pressure for a sample individual is 104.76 mmHg, with a 95% prediction interval of 103.74–105.79 mmHg, demonstrating good model fit.
